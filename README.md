# Теоритическая база
Дилатация - одна из фундаментальных операций математической морфологии, которая "расширяет" или "утолщает" объекты на бинарном изображении. При дилатации структурный элемент перемещается по изображению и в итоговом изображении 1 записывается в тех случаях, когда совпали хотя бы 1 пиксель изображения и структурного элемента.

# Описание разработанной системы
Функция native_dilate - нативная реализация дилатации с помощью numpy.
Функция library_dilate - реализация дилатации с помощью openCV
